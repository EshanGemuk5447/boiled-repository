# boiled-repository
Boiled Repository 

# DOWNLOAD
```curl -sSL https://raw.githubusercontent.com/EshanGemuk5447/boiled-repository/main/boiled-ft.sh | sudo bash```

# USE
type ls $boiled to see all items, then if you see one (example: @fastfetch) then ls $boiled/@fastfetch then you'll see the package (example: fastfetch)
then type sudo boil install @fastfetch/fastfetch

# REMINDER
We would kindly to remind you to always see this repository because we update the repository file always. And if you see the repository file name updated, run
```sudo rm -rf /usr/local/bin/boil /boiled``` then relaunch the install method code
